# Documentação Complementar

Esta pasta armazena documentação técnica e de negócio relacionada ao projeto de triagem automatizada.

## Conteúdo sugerido

- **Glossário técnico** — vocabulário específico das verticais Arrecadação, Atendimento e ISS.
- **Exemplos de chamados** — exemplos anonimizados de chamados-padrão e seus comentários internos esperados.
- **Regras de negócio** — anotações sobre leis, decretos e regras tributárias recorrentes (ISS, IPTU, ITBI, taxas etc.).
- **Mapeamento de produtos** — relação entre os produtos (Tributos, Procuradoria, Gestão Fiscal, e-Nota, Cidadão Web, Livro Eletrônico, Protocolo) e seus principais módulos/erros conhecidos.
- **Decisões de projeto** — registros (ADRs) sobre mudanças de comportamento da triagem ao longo do tempo.

> Esta pasta é versionada no Git e seu conteúdo serve como fonte de consulta para a IA e para os analistas.
